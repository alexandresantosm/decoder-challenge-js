# Decoder - Decodificador de Texto

> Um decodificador de textos feito com HTML5, CSS3 e JavaScript.

## Sobre

Aplicação que criptografa textos, assim é possível trocar mensagens secretas com outras pessoas que saibam o segredo da criptografia utilizada.

## Ferramentas

- Editor de texto qualquer, utilizei o [Visual Studio Code](https://code.visualstudio.com/) durante o desenvolvimento.

- Navegador Web, utilizei o [Firefox](https://www.mozilla.org/pt-BR/firefox/new/) durante o desenvolvimento.

## Tecnologias

A aplicação Decoder foi desenvolvida utilizando as seguintes tecnologias mais comuns no desenvolvimento Web:

- HTML5
- CSS3
- JavaScript
